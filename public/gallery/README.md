# Imagens da Galeria

Adicione as seguintes imagens nesta pasta:

## Estrutura esperada:

```
public/
  gallery/
    dj-mixer.jpg      - Imagem grande do DJ com as mãos no mixer
    dj-setup.jpg      - Vista lateral com laptop e equipamento
    dj-hands.jpg      - Close-up das mãos do DJ no equipamento
```

## Como adicionar:

1. Salve as 3 imagens fornecidas nesta pasta com os nomes acima
2. As imagens aparecerão automaticamente na galeria

## Formatos recomendados:

- JPG/JPEG para melhor compressão
- Dimensões: Mínimo 800x600px para melhor qualidade
- Tamanho do arquivo: Máximo 500KB por imagem
